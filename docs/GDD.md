# 游戏设计方案：今晚不飞升 (No Ascension Tonight)

## 1. 核心概念 (Core Concept)
《今晚不飞升》是一款结合领地生存、防守压力与高频战斗成长的修仙 Roguelike。玩家是一名踏入山海秘境的修仙者，需要守住中央的“灵核/道基”，在昼间斩杀四方妖兽、掠夺四色灵性，夜间守住山门，最终在第 20 波完成飞升。

## 2. 设计支柱 (Design Pillars)
1. **方向即策略**：四个方向对应四条资源路线、四种敌人生态、四个大妖。
2. **平衡有收益，偏科有代价**：四色最低值决定基地光环半径。
3. **进攻服务防守**：昼间行动决定夜间怪潮强度和路线。
4. **斩首大妖的时机**：全局决策的核心。

## 3. 核心循环 (Game Loop)
1. **昼：远征与削弱 (60s)**：远征斩妖，掠夺灵性。
2. **转场：商店与建设 (20-30s)**：购买法宝、强化阵法、修补灵核。
3. **夜：阵地防守 (60s)**：在灵核光环内抵御妖兽冲击。
4. **飞升 (波次 20)**：终局挑战。

## 4. 基地系统：灵核 (The Life Core)
- **护山灵光 (Radiation Range)**：主场加成区。
- **木桶原理 (Lowest-Level Gating)**：光环半径由四色灵性中最低等级决定。

## 5. 四大山门与大妖
- **北 (红)**：近战冲锋，削弱大妖攻击力。
- **南 (绿)**：生命回血，削弱大妖最大生命。
- **东 (蓝)**：远程弹幕，削弱大妖攻击频率。
- **西 (黄)**：高速位移，削弱大妖移动速度。

## 6. 斩首行动 (The Beheading)
提前挑战大妖，成功后封死对应山门，减少夜间压力，但该色能量产出将受限。
